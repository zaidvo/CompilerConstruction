# CalcScript++ Compiler - Implementation

## Quick Start

### Launch GUI (Recommended)

```bash
python launch_modern_gui.py
```

Or on Windows, double-click: **`Launch_Modern_GUI.bat`**

### Command Line Interface

```bash
python main.py <source_file.calc>
```

## Files

- **`launch_modern_gui.py`** - GUI launcher
- **`Launch_Modern_GUI.bat`** - Windows batch launcher
- **`main.py`** - Command-line interface
- **`gui_modern/`** - Modern GUI package with purple-gold-teal theme
